\section{Dữ liệu và tiền xử lý}

\subsection{Nguồn dữ liệu}

Dữ liệu gốc là file \texttt{WDIEXCEL.xlsx} của World Bank, gồm các sheet chính \texttt{Data}, \texttt{Country}, \texttt{Series} và các bảng metadata liên quan. Nhóm sử dụng Python để đọc trực tiếp file Excel gốc, xác minh các mã indicator cần dùng trong sheet \texttt{Series}, đồng thời lấy thông tin quốc gia, khu vực và nhóm thu nhập từ sheet \texttt{Country}.

Vì file WDI chứa cả quốc gia thật và nhiều dòng tổng hợp như World, nhóm thu nhập hoặc khu vực tổng hợp, nhóm chỉ giữ các đơn vị có trường \texttt{Region} hợp lệ trong sheet \texttt{Country}. Quyết định này giúp mỗi quan sát trong dữ liệu đầu ra tương ứng với một đơn vị địa lý cụ thể, phù hợp với các biểu đồ so sánh quốc gia và dashboard trong Tableau.

\subsection{Quy trình tiền xử lý chung}

Thay vì để từng mục tiêu đọc và làm sạch riêng từ file gốc, nhóm xây dựng một pipeline tiền xử lý chung. Pipeline này tạo trước một file dữ liệu sạch nền tảng, sau đó các file dữ liệu theo mục tiêu được trích từ chính file sạch chung đó.

Quy trình gồm các bước:

\begin{enumerate}
    \item Đọc file \texttt{WDIEXCEL.xlsx} và metadata từ các sheet \texttt{Data}, \texttt{Country}, \texttt{Series}.
    \item Lọc các indicator thuộc bốn mục tiêu phân tích: kinh tế, giáo dục, sức khỏe--dân số và môi trường.
    \item Chỉ giữ giai đoạn 2000--2023 và các quốc gia hoặc vùng lãnh thổ có \texttt{Region} hợp lệ.
    \item Với từng cặp \texttt{Country\_Code} và \texttt{Indicator\_Code}, loại chuỗi không có quan sát hoặc có khoảng thiếu nội bộ dài hơn 2 năm liên tiếp.
    \item Nội suy tuyến tính các khoảng thiếu nội bộ dài tối đa 2 năm; không ngoại suy ở đầu hoặc cuối chuỗi thời gian.
    \item Xuất dữ liệu sạch chung dạng long format thành \texttt{data\_output/wdi\_clean\_base.csv}.
    \item Từ file sạch chung, pivot và xuất các file wide format riêng cho từng mục tiêu để kết nối vào Tableau.
\end{enumerate}

Cách tổ chức này giúp nhóm vừa có một chuẩn làm sạch thống nhất, vừa vẫn giữ được cấu trúc dữ liệu riêng phù hợp với từng nhóm biểu đồ trong Tableau.
